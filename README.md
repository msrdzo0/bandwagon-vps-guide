## 搬瓦工怎么使用：从注册到 SSH 连接的完整步骤

### 第一步：注册账号

打开搬瓦工官网注册页面，填写邮箱、设置密码，注意几点：

- 邮箱要填真实的，后续 VPS 信息会发到这里
- 国家/地区选自己实际所在地，影响到后续的支付选项
- 注册完成后会发一封验证邮件，点击确认

### 第二步：选套餐、下单

1. 进入套餐页面，选好你要的方案
2. 选择机房（没有特殊需求，先选洛杉矶 DC6 CN2 GIA-E 或 DC9）
3. 选择计费周期（季付、半年付、年付，越长单价越低）
4. 点击 "Add to Cart" 进入结账页面
5. 结账页有一栏 "Promotional Code"，可以填搬瓦工官方活动促销码
6. 选支付方式——支持支付宝，直接扫码付款，国内用户很方便

### 第三步：获取 VPS 连接信息

付款成功后，系统会发邮件告诉你 VPS 已经创建。

进入后台操作方式：

1. 登录搬瓦工账号，点击右上角 "Client Area"
2. 进入 **Services → My Services**，就能看到你买的 VPS
3. 点击对应 VPS 后面的 **"KiwiVM Control Panel"** 按钮
4. 进入 KiwiVM 面板后，首页就能看到 **Public IP address** 和 **SSH Port**
5. root 密码需要手动生成：点击左侧菜单 "Root password modification" → "Generate and set new root password"，密码只显示一次，记下来

到这里，你有了三样东西：**IP 地址、SSH 端口号、root 密码**。可以连接了。

### 第四步：用 SSH 工具连上去

**Windows 用户：**

推荐用 Xshell（有免费版），打开后新建会话，填入 IP 地址、端口号，用户名填 `root`，密码就是刚才记下来的那个。

也可以直接用 PowerShell，命令格式：


ssh root@你的IP地址 -p 你的SSH端口


**Mac/Linux 用户：**

直接开终端，命令一行搞定：


ssh root@你的IP地址 -p 你的SSH端口


回车后输入密码，输入时不显示字符，这是正常的，输完直接回车就行。

**看到命令行出现 `[root@localhost ~]#` 就说明连进去了。**

这时候你面对的就是一台完整的 Linux 服务器，从这里开始你能做的事情就多了。

## 常见问题

**Q：搬瓦工支持哪些付款方式？**

支持支付宝、PayPal、信用卡、银联，国内用户用支付宝最方便，买的时候选一下就行，扫码付款，几秒钟完成。

**Q：搬瓦工怎么使用时，SSH 连接老是断怎么办？**

先检查本地网络。如果网络没问题，SSH 断连很可能是服务器端 SSH 空闲超时设置的问题，可以在服务器里修改 `/etc/ssh/sshd_config`，增加 `ServerAliveInterval` 和 `ClientAliveInterval` 配置。另外，如果你在国内访问，晚高峰时段线路拥堵也会导致延迟上升甚至掉线，这时候可以用 KiwiVM 尝试迁移到更近的机房。

**Q：限量版套餐和普通套餐有什么区别？**

限量版通常配置更高、价格更低，但经常缺货，补货后几小时就卖完。常规套餐随时可买，但单价相对高。如果不急，可以等限量版补货；如果现在就要用，直接上常规套餐。

**Q：搬瓦工怎么重装系统？**

进入 KiwiVM 面板 → 左侧菜单找 "Install new OS" → 选择你想要的 Linux 发行版 → 点击确认。重装完成后会显示新的 root 密码，记得截图保存。整个过程大概几分钟。

**Q：KiwiVM 面板登录不进去怎么办？**

搬瓦工后台启用了 Cloudflare 验证，有时候会出现验证卡住的情况。多刷新几次通常能过。如果持续登录不了，可以尝试换个浏览器或者清一下 Cookie。
